# Vue3 基础框架

一个使用 Vite + Vue3 组合式API 搭建的纯前端基础框架。

## 技术栈

- Vite 5.0
- Vue 3.4 (Composition API)
- 纯 CSS

## 项目结构

```
src/
├── main.js             # 入口文件
├── App.vue             # 主组件
├── components/         # 组件目录
├── composables/        # 组合式函数
└── styles/             # 样式文件
```

## 功能特点

- ✅ 组合式API（Composition API）
- ✅ ref 状态管理
- ✅ 基础页面布局
- ✅ 简洁样式
- ✅ 预留大量扩展空间

## 启动项目

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 构建项目

```bash
# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
